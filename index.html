Vakken

Codelab

Bij codelab ga je een uurtje of meer werken aan de opdrachten die je krijgt van school. In dit uur kan je ondersteuning krijgen van de docenten indien je dit nodig hebt. Je kan het vergelijken met een huiswerk uur op de middelbare.

Windows development

Bij Windows development leer je in het eerste jaar Python en C#. De hashtag spreek je niet uit als hashtag, maar als sharp. Het is dus C sharp. Het is niet alleen voor windows, omdat je natuurlijk deze talen ook hebt voor iOS. Daarom is de naam ook veranderd naar Native. Dit houd dus in dat op elk platform met een bepaalde taal kan programmeren.

Web development

Bij web development leer je alles wat te maken heeft met web, denk hier aan HTML, CSS, JS, Laravel en Php. In het begin van het jaar leer je de basic van html en de basic van css. In het begin leer je het eigenlijk de moeilijke manier. Je moet dus bijvoorbeeld echt alles zelf gaan maken terwijl er op internet al voorbeelden staan die je eventueel zou kunnen gebruiken.

Generieken vakken

Engels

Bij Engels begin je eigenlijk gelijk met opdrachten maken, als je een opdracht blok hebt afgerond maak je een toets. Je moet voor elk blok een toets hebben gemaakt wil je voor examen op gaan voor het vak.

Nederlands

Bij Nederlands is het eigenlijk het zelfde verhaal, je maakt opdrachten en krijgt ook uitleg in de les. Je gaat ook heel veel bezig zijn met teksten schrijven. Dit zijn dan zakelijke teksten, zodat je eraan kunt wennen om zakelijk te gaan schrijven.

Rekenen

Bij Rekenen begint de les met uitleg over wat je moet gaan doen en daarna moet je opdrachten gaan maken. Ook bij dit vak moet je alle blokken hebben afgerond wil je opgegeven worden voor examen.
